Text Prediction App

Wagner Pinheiro
March, 2017

JHU Data Science Specialization Capstone Project

The Project

This app uses a predictive model using Markov Chains to suggest the next user word. This model was defined from the analysis of n-grams in a collection of texts extracted from blogs, twitter and news.

The texts dataset used can be accessed here.
